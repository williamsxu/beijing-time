# 北京时间网站 - Vue3复刻版

这是一个使用Vue 3、Bun和TailwindCSS复刻的北京时间网站。该项目展示了实时的北京标准时间，以及相关的时间工具和世界时间查询功能。

## 功能特点

- 实时北京标准时间显示
- 日期和农历信息
- 时间工具导航
- 世界时间查询导航
- 响应式设计，适配不同设备

## 技术栈

- Vue 3 - 前端框架
- Bun - 构建工具和包管理器
- TailwindCSS - CSS框架
- Vite - 开发服务器和构建工具

## 安装和运行

```bash
# 安装依赖
bun install

# 开发环境运行
bun run dev

# 构建生产版本
bun run build

# 预览生产构建
bun run preview
```
